# CineBot AI - NLP Movie Recommendation Chatbot

An intelligent movie recommendation chatbot built with advanced NLP techniques.

## Features
- TF-IDF Vectorization
- Cosine Similarity Matching
- Sentiment Analysis
- Beautiful Gradio UI

## Demo
Open in Colab: https://colab.research.google.com/github/786jabar/CineBot-AI/blob/main/CineBot_Colab.ipynb

## Technologies
- Python
- Scikit-learn
- TextBlob
- Pandas
- Gradio

## Example Queries
- Funny comedy movie
- Scary horror film
- Romantic movie
- Action thriller
